============
Contributing
============


Critiques are Welcomed!
-----------------------

Conventions are of little use if no is interested in adopting them, and the best way to ensure
adoption is by involving as many users as possible as early as it is feasible. Therefore,
this work is being made public in an unfinished, Work-in-Progress state so others
can participate in it's creation.

A knowledge of python is helpful, but users unfamiliar with coding can still browse the :ref:`Convetions <conventions>` 
and :ref:`Examples <common_errors>` sections and provide feedback.

Getting Involved
----------------

If interested, the best way to collaborate is to create a
`Pull Request <https://github.com/argerlt/Consistent-Crystal-Conventions/pulls>`_ through GitHub, which allows
for collaborative development before eventually merging into the main project. 

Alternatively, emails can also be sent to `argerlt@gmail.com <argerlt@gmail.com>`_, 

Regardless of method, if your input changes the code, you will be credited as an author for this project.
